## 設計思路

> 現今社會，人們的生活節奏很快。每時每刻，人們都在被工作，學業功課，和 周圍不間斷更新的事物所追趕。無法慢下腳步，放鬆身心。
> “日常生活中，一個人們可以休息片刻，提供悠閒安靜環境的地方有什麼呢？”
> 我們想到了“咖啡館”，一個有地道好喝的咖啡搭配好吃的糕點，在溫馨安靜的環境中，適合看書，聽音樂，三五好友 一起坐下來聚會聊天，有情調又小資的地方。
